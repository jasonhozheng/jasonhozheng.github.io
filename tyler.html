Tyler The Creator

Introduction

This website is a short project done during BCIT's Technical Web. My goal was to create a website which is meant to promote the artist. It has simple navigation with to showcase a biography and tracks available to listen to by Tyler.

I wanted to create a website which was simple didn't have to many pages, to mainly showcase the about and tracks pages. I used vertical headings to bring a more unique look into the simple but clean design. My goal with the coloring was to use a much darker color pallete to bring out the brighter pictures and font.

Development Process

The site was built with HTML and CSS. Starting with the main content I already knew what colors I had wanted to use so it was less envisioning and more of just putting what I had wanted to an actual website. I started with the base HTML and features before moving into the placement and positioning of the content.
